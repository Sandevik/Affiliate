Här kan du lägga till guide sidor om du vill ha.

Skapa en fil som heter "filnamn".tsx, Ex: råd-om-omstartslån.tsx.

lägg till detta i den tomma filen:

import Head from 'next/head'
import React from 'react'
import StandardContentBlock from '../../components/Blocks/StandardContentBlock'
import SubHero from '../../components/SubHero'
import { useTheme } from '../../context/ThemeContext'
import style from "../../styles/Page.module.css"

export default function Index() {
  const {darkMode} = useTheme()
  return (
    <>
    <Head>
        <title>Guide om x | Omstartslånet</title>
        <meta name="description" content="Generated by create next app" />
        <meta name="robots" content="index, follow" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <link rel="icon" href="/favicon.ico" />
    </Head>

    <div className={darkMode ? `${style.container} ${style.darkmode}` : style.container}>
      <SubHero>
        <h1>en guide om ...</h1>
        <p>Här är allt du behöver veta </p>
      </SubHero>
    </div>
    </>
  )
}

Ändra och lägg till tills det blir som du önskar i både Head taggen och i diven som är under Head taggen.

OBS! Om du vill skapa egna css klasser måste du skriva className istället för class, 
då class redan är reserverat i javascript. 
Sedan måste det stå {style."klass namn"},
sedan kan du skriva alla klasser i filen /styles/Page.module.css
